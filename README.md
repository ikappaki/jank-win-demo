# jank-win-demo

A [jank](https://jank-lang.org) MS-Windows demo showcasing native Win32 interop and message handling.

It creates a window with bouncing text. Clicking the window changes the text color and updates the title.

## Requirements

Install **jank** on Windows by either:

- Using a prebuilt release:
  https://github.com/ikappaki/jank-win-release

- Building from source:
  https://github.com/ikappaki/jank-win

## Run

Once `jank` is installed:

```bash
jank run demo.jank
```
